# HaloPNJ

## Overview

The AI Chatbot Monitoring Dashboard is an intuitive web application designed to empower non-technical users to maintain, monitor, and optimize their AI-powered chatbot systems. This platform provides comprehensive analytics, automated question clustering, and real-time performance metrics to users understand and improve their chatbot interactions.

## Project Objectives

- **User-Friendly Monitoring**: Enable non-IT users to effectively monitor AI chatbot performance
- **Automated Insights**: Provide intelligent clustering of frequently asked questions using advanced ML techniques
- **Performance Tracking**: Monitor chatbot usage patterns and user engagement metrics
- **Maintenance Simplification**: Streamline chatbot maintenance and optimization processes

## Key Features

### Chatbot Analytics Dashboard
- **Performance Monitoring**: Monitor chatbot accuracy and user satisfaction scores
- **Trend Analysis**: Identify usage patterns and peak interaction times

### FAQ Clustering & Analysis
- **UMAP Dimensionality Reduction**: Advanced technique to visualize high-dimensional question data
- **HDBSCAN Clustering**: Automated grouping of similar questions into meaningful categories
- **Interactive Visualization**: Explore clustered questions through intuitive visual representations
- **Topic Identification**: Automatically detect and label common conversation themes

## Technology Stack

### Frontend
- **Vue.JS** - Modern UI framework
- **Chart.js** - Analytics charts and graphs
- **Tailwind CSS** - Responsive styling

### Backend
- **Node.JS with Express.JS** - Responsive styling
- **Python** - Core programming language
- **FastAPI** - High-performance web framework
- **UMAP** - Uniform Manifold Approximation and Projection for dimensionality reduction
- **HDBSCAN** - Hierarchical Density-Based Spatial Clustering
- **Scikit-learn** - Additional ML utilities and preprocessing

### Database & Infrastructure
- **PostgreSQL** - Primary database for storing chat logs and analytics
- **Docker** - Containerization for easy deployment
